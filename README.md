# CVR 基于深度学习的虚拟图像到真实图像的转换
开发平台：Pycharm
环境需求：Linux、Python3、Anaconda虚拟环境配置工具、CPU or NVIDIA GPU + CuDNN
虚拟环境需要安装的软件包：
1.	Python==3.9
2.	torch==1.11.0
3.	numpy==1.24.3
4.	torchvision
5.	mpi4py（最好使用conda安装）
6.	pytorch-cuda==11.8
7.	tqdm
8.	omagaconf
9.	pandas
10.	pillow
11.	opencv_python
12.	einops
13.	basicsr==1.3.4.9
注：虚拟环境创建指令为 conda create -name 自主命名环境 python==3.9
